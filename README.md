# 💱 CurrencyBot

A Telegram bot for real-time currency conversion.

## ✨ Features

- Convert between 8 popular currencies
- Fully button-based interface — no manual input needed
- Live exchange rates via [ExchangeRate API](https://exchangerate-api.com)
- Shows the rate for 1 unit of the base currency

## 💬 Supported Currencies

🇺🇸 USD · 🇰🇿 KZT · 🇪🇺 EUR · 🇨🇳 CNY · 🇷🇺 RUB · 🇨🇭 CHF · 🇬🇧 GBP · 🇹🇷 TRY

## 🛠 Tech Stack

- Python 3.13
- python-telegram-bot 22.7
- httpx
- ExchangeRate API
- Deployed on Railway
